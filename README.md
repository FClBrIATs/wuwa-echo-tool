# wuwa-echo-tool
装備中音骸の厳選優先度をスコアで可視化し、新規音骸と比較することが出来る私的ツールです。 単一HTMLファイルで完結し、インストール不要・サーバー不要でブラウザから即使用できます。

## 機能一覧

### ① ステータス入力タブ
- キャラの攻撃力・HP・防御力・クリ率・クリダメ・各属性ダメバフ・スキル別ダメバフを入力
- プリセット収録済みのハーモニーを最大2枠（2セット×2 or 手動）から選択し、バフを自動加算
- 武器・キャラ登録からの自動加算（登録済みデータをドロップダウンで選択）
- 自動加算ON/OFFトグル、「自動」行で各バフの内訳を確認可能
- ステータス優先度ランキング：現在のビルドに対して「攻撃力%/クリ率/クリダメ/属性ダメバフ/スキルバフ」のどれが最も期待ダメージ向上に寄与するかを表示

### ② 装備音骸タブ
- 装備中の音骸5スロットのコスト・メインステ・サブステ（最大5枠）を入力
- 各スロットのスコア（① のステータスを基準にした期待ダメージ貢献度）を計算・比較
- 最弱スロットを赤枠、最強スロットを緑枠でハイライト表示
- メインステはコスト選択で対応する値が自動決定・①タブのバフ計算に反映

### ③ 新規判定タブ
- 新規入手した音骸のサブステを入力し、装備中スロット（自動 or 指定）の合格ラインと比較
- 残サブ枠で合格ラインを超える期待確率を正規分布近似で算出

### ⚙ 登録タブ（ハーモニー / 武器 / キャラ）
- ハーモニー登録：未収録のハーモニーを手動で追加し、①タブの選択肢に反映（2+5セット・3セット対応）
- 武器登録：基礎攻撃力・各バフ値を登録してドロップダウンから選択適用
- キャラ登録：キャラ固有の能力値・バフ等を登録して自動加算

---

## 使い方

1. `echo-optimizer-v5.html` をダウンロード
2. ブラウザで開く（Chromeを推奨）
3. ① タブでキャラのステータスを入力 → ② タブで装備音骸を入力 → ③ タブで新規音骸を判定

> データの保存：入力データはブラウザの `localStorage` に自動保存されます。ブラウザのデータ消去・シークレットモードでは保持されません。

## 技術仕様

- **バニラ HTML / CSS / JavaScript**（フレームワーク・ライブラリ不使用）
- **単一ファイル**完結（外部リソースはGoogleフォントのみ）
- データ永続化：`localStorage`
- スコア計算：サブステ種別ごとの**限界価値（Marginal Value）**をベースにした期待ダメージ貢献度
- 確率計算：残サブ枠の正規分布近似
